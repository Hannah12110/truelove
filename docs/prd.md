# 恋爱时机测试需求文档

## 1. 应用概述

### 1.1 应用名称

恋爱时机测试

### 1.2 应用描述

这是一个专业的心理测试网页,通过20道情境化题目评估用户的恋爱准备状态,帮助用户了解自己的恋爱时机,并提供心理学解读和具体行动建议。

## 2. 核心要求

### 2.1 情境化题目

测试题目通过具体场景和选择来测试用户的潜意识反应,避免直白的自我评价。

### 2.2 互动性强

测试过程要有参与感,通过动画效果和交互反馈提升用户体验。

### 2.3 结果有深度

测试结果包含类型判定、雷达图展示、详细文字解析和3个具体行动建议。

### 2.4 视觉美观

采用浪漫温馨的设计风格,适合女性用户,符合小红书审美。

## 3. 页面结构

### 3.1 欢迎页

- 大标题:恋爱时机测试
- 副标题:测测你的正缘什么时候出现?
- 测试说明:20道情境化题目,3分钟了解你的感情状态
- 开始测试按钮(醒目,带动画效果)
- 底部免责声明:本测试仅供娱乐,非专业心理测评

### 3.2 测试页

- 顶部进度条(实时显示:第X题/20题)
- 每题包含:
  - 题目描述(场景化,有代入感)
  - 4个选项(A/B/C/D标注,每个选项有简短说明)
  - 下一题按钮(点击后自动进入下一题,有平滑过渡动画)
- 侧边提示:请根据你的第一反应选择答案

### 3.3 结果页

- 雷达图展示4个维度的得分(时机已到、等待中、需要调整、享受单身)
- 显示主要结果类型(大标题+图标)
- 详细文字解析(核心特征、优势、2025年爱情运势)
- 3个具体行动建议(可点击展开详情)
- 分享按钮(生成结果海报/复制结果文字)
- 重新测试按钮(清除答案,重新开始)

## 4. 设计风格

### 4.1 配色方案

- 主背景:浅粉色渐变(#FFE4E1 → #FFB6C1 → #FFF0F5)
- 强调色:树莓深粉色(#E63C6E)
- 文字颜色:深灰色(#333333)
- 次要文字:浅灰色(#888888)
- 按钮背景:树莓深粉色,带阴影效果
- 高亮色:金色(#FFD700,用于星级评分)

### 4.2 字体

- 主标题:圆润可爱的字体(如思源黑体 Bold或阿里巴巴普惠体 Bold")
- 正文:清晰易读的字体(如思源黑体 Regular")
- 装饰性文字:手写体(用于情感化表达)

### 4.3 视觉元素

- 爱心、星星、云朵等浪漫元素(淡粉色,半透明)
- 柔和的阴影和渐变效果
- 圆角设计(按钮、卡片、容器都有圆角)
- 微妙的动画效果(渐入、渐出、缩放)

### 4.4 布局

- 移动端优先(竖屏适配)
- 居中对齐,内容紧凑但不拥挤
- 充足的留白,营造舒适感

## 5. 题目内容

### 情境1:社交场景

第1题:在朋友的聚会上,你注意到有人一直在看你。你会:

A. 假装没看见,继续和朋友聊天
B. 主动走过去打招呼
C. 心里暗暗期待他会过来,但不敢主动
D. 觉得很不自在,想早点离开

第2题:朋友给你介绍了一个对象,发来对方的照片。你的第一反应是:

A. 立刻翻看对方的朋友圈,了解更多信息
B. 有点紧张,不知道该聊什么
C. 觉得"还凑合"，但也没有特别心动

D. 没什么兴趣，觉得自己现在不需要


**3. 你在社交媒体上刷到一个很喜欢的博主，Ta 的恋爱观和你很像。你会：**A. 立刻关注，并想和 Ta 做朋友B. 点赞几条内容，但不会主动互动C. 觉得"说得好"，但不会特别在意D. 无感，划过去

### 情境 2：情感反应

**4. 看到朋友晒出甜蜜的合照，你的第一反应是：**A. 真心地为他们感到开心B. 有点羡慕，也想有一段这样的关系C. 觉得"挺甜"，但没有强烈的情绪波动D. 甚至有点反感，觉得秀恩爱很无聊

**5. 听到朋友分手的消息，你会：**A. 主动安慰，愿意听 Ta 倾诉B. 感慨"感情真复杂"，心里有点难过C. 觉得"可惜"，但很快就过去了D. 觉得"这很正常"，不太受影响

**6. 看到一部爱情电影，结尾男女主角终于在一起。你会：**A. 感动落泪，觉得爱情很美好B. 暗暗期待自己也能遇到这样的爱情C. 觉得"挺好看的"，但不会特别激动D. 觉得"太戏剧化了"，不太现实

### 情境 3：自我认知

**7. 如果用三个词形容你现在的感情状态，你会选：**A. 充满期待、准备好了、相信爱情B. 渴望但有点害怕、等待对的人、不将就C. 有些困惑、需要时间、先提升自己D. 享受当下、独立自主、不着急

**8. 你觉得，一段好的感情最重要的条件是：**A. 双方都愿意付出，共同成长B. 彼此理解和支持，给对方安全感C. 合适的时机，对的人D. 不强求，顺其自然

**9. 如果现在有一段感情向你靠近，你的态度是：**A. 勇敢迎接，相信缘分B. 既期待又有点紧张，慢慢来C. 观察一段时间，再决定D. 觉得"还没准备好"，可能会拒绝

### 情境 4：成长轨迹

**10. 回想过去的一段感情，你觉得最大的收获是：**A. 明白了自己真正想要什么B. 学会了如何更好地爱人C. 发现了自己的不足，需要改进D. 意识到单身其实也很好

**11. 你对未来 3 年的感情期待是：**A. 希望能遇到对的人，走向稳定B. 保持开放心态，如果遇到了就试试C. 先专注于自己，感情随缘D. 不刻意期待，也不排斥机会

**12. 如果现在没有感情，你会：**A. 积极参加社交，扩大机会B. 偶尔会期待，但不会特别主动C. 把更多时间放在自己身上D. 觉得很自在，享受单身生活

### 情境 5：现实压力

**13. 家人开始催婚，你的反应是：**A. 会认真考虑，但不会因为催促而妥协B. 有点压力，但觉得"缘分到了自然会遇到"C. 觉得"还早"，不想着急D. 不太在意，觉得单身也挺好的

**14. 看到同龄人结婚生子，你会：**A. 真心祝福，同时期待自己的缘分B. 有点羡慕，但不会特别焦虑C. 觉得"每个人都有自己的节奏"D. 觉得"这样也不错，但我还是更自由"

**15. 如果一直遇不到对的人，你会：**A. 相信缘分，继续寻找B. 可能会降低一些标准，试试看C. 觉得"可能缘分没到"，先过好自己D. 觉得"单身也很好"，不一定非要恋爱

### 情境 6：内心冲突

**16. 你觉得"对的人"应该：**A. 彼此有相似的价值观和目标B. 能给你安全感和归属感C. 合适的时机出现，自然相遇D. 不强求，如果遇到了就是缘分

**17. 你在感情中最害怕的是：**A. 再次受伤害，不敢全身心投入B. 遇不到对的人，浪费时间C. 选择了错误的人，后悔D. 失去了自由，被束缚

**18. 如果有人说"你现在不适合恋爱"，你会：**A. 有点受伤，但会反思自己B. 觉得"可能对吧"，慢慢调整C. 觉得"这是我的选择"，不会太在意D. 完全不认同，单身也很好

### 情境 7：行动倾向

**19. 假设你对一个朋友有了好感，你会：**A. 找机会多接触，试探对方的意思B. 有点纠结，不知道该不该主动C. 先观察一段时间，再做决定D. 压抑这种感情，保持朋友关系

**20. 如果你现在必须给自己一个"感情状态标签"，你会选：**A. 渴望爱情，准备好了B. 期待爱情，但还在等待C. 有期待但不急，先提升自己D. 享受单身，不着急
